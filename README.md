# Social Media Demo-Platform

A full-stack social media platform built using the MERN Stack.

## Features

- User Authentication
- User Profiles
- Create Posts
- Like Posts
- Comment System
- Follow / Unfollow Users
- Responsive Design

## Tech Stack

Frontend:
- React.js
- React Router
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- JWT Authentication
